# PSEUDOCÓDIGO
Es una forma de escribir un programa utilizando palabras más sencillas, lo que ayuda a entender la estructura de un código de programación y poder aprender cualquier otro lenguaje.

## ALGORITMOS
### ROMANOS
El pseudocódigo de los numeros romanos fue complicado ya que al inicio es complicado entender la forma como el programa entra en los arreglos
